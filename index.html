HTML Forms Reference


Form Tags


Tag Name Description
<form> Form It is used to collect information and sending it to its appropriate location for processing
<input> Input This is used to add fields for users to input information in relation to their specific types such as: texts, numbers, passwords, range, checkbox, radio [button], list and submit
<label> Label This is used to label the identity of each <input>. For example: If you go to a shopping mall, you'll see that there are lots of goods properly arranged in different sections, and each section is labelled according to the class of goods it represents, like beverages, books, toiletries, etc.
<select> Select This is used to create a dropdown list to select a listed item
<datalist> Datalist Similar to <select> tag, this also creates a dropdown list with the ability to type in the name of the item or information you want, unlike the <select> tag that doesnt have such functionality
<textarea> Textarea The textarea is a field used for inputing texts, and can be customized to take in a limited amount of text characters
